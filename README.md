| # | Process |
| ------ | ------ |
| 1 | Enter locations. (SOUTH CAROLINE, NEW YORK, WASHINGTON ETC…) |
| 2 | Search **(MUST CHROME BROWSER)** |
| 3 | Send E-Mail **(MUST OUTLOOK)** |
